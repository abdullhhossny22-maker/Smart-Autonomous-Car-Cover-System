##  Test 1: Ultrasonic Sensor
### Objective
The goal of this test is to see if the ultrasonic sensor works and if it can measure distance correctly.
### Procedure
To test the sensor I did the following steps:
- I connected the ultrasonic sensor to the Arduino board.
I connected the VCC to 5V on the Arduino.
I connected the GND to the GND on the Arduino.
I connected the Trig to Pin 9 on the Arduino.
I connected the Echo to Pin 10 on the Arduino.
- Then I uploaded the code to the Arduino to measure the distance using the sensor.
- After that I opened the Serial Monitor to see the results.
### Result
The results were good.
- The ultrasonic sensor displayed the distance values correctly.
- When I moved an object closer or farther, from the sensor the readings changed.
### The ultrasonic sensor is working the way it should. The ultrasonic sensor test was successful.

##  Test 2: LDR Sensor (Light Detection)
### Objective
The goal of this test is to see if the LDR sensor works and to check the light intensity readings. I want to know if the LDR sensor can detect light.
### Procedure
To test the LDR sensor I did the following steps:
I connected the LDR sensor using a voltage divider.
I connected one side of the LDR sensor to 5V.
I connected the side of the LDR sensor to A0.
I also connected a resistor between A0 and the ground.
- Then I uploaded some code to the system to read the analog values from the LDR sensor.
- I used Tinkercad to change the level and see what happens.
### Result
What I found out is that the light values changed when the light intensity changed.
- When it was dark the LDR sensor produced values.
- When it was bright the LDR sensor produced values.
The LDR sensor is able to detect intensity.
### The LDR sensor is working the way it should. I am  happy, with the result of the LDR sensor test. The LDR sensor test was a success.
